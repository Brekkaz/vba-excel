cada reporte se diligenciara desde el archivo 'Formato Reporte',
se debe guardar desde la opcion 'Guardar Como'
y su nombre debe comenzar por 'co'

el resporte se generara desde el archivo 'Reporte'
el cual analizara solamente los reportes que se encuentren
en la carpeta donde se encuentre este
y que comienzen con las letras 'co'

nota:
1.los campos que se dejen vacios o sean de unica seleccion
y se selccione mas de una opcion se rellenaran con datos default
2.para desselecionar las opciones escogidas con 'x'
se debe borrar el texto como en una casilla cualquiera
